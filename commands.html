Available Commands¶

cat command¶

class commands.cat.CatCommand(context, log, get_input)

Prints the file to terminal. If ‘-w’ option is specified, the cat command prompts the user for input and overwrites the file with that input.

Usage: cat [OPTIONS] <path_to_file>

cat-at command¶

class commands.cat_at.CatAtCommand(context, log, get_input)

Without -w option, cat-at reads the content which starts at the specified index with a specific size (which is taken as a separate input). With the -w option, it appends the content that is logged by the user separately at the specified index.

Usage: cat-at <index> <fileName> cat-at -w <index> <fileName>

cat-end command¶

class commands.cat_end.CatEndCommand(context, log, get_input)

Appends user input at the end of the given file name.

Usage: cat-end <filename>

this command prompts the user to input the text, by displaying “Start Writing: “. Whatever is entered is appended to the <filename>.

commands.cd command¶

class commands.cd.CDCommand(context, log, get_input)

Used to change the directory in which the user resides in, to one of the constituents directories.

Usage: cd <directory_name>

commands.ls command¶

class commands.ls.LSCommand(context, log, get_input)

Print out the files and directories that presently exist in the directory user currently resides in.

Usage: ls

commands.mkdir command¶

class commands.mkdir.MkDirCommand(context, log, get_input)

Used to make a new directory inside the directory user presently exists in.

Usage: mkdir dir_1

commands.mv command¶

class commands.mv.MvCommand(context, log, get_input)

Moves the given file or directory to the given target directory. Raises an error if the file/directory to be moved does not exist.

Usage: mv <filename1> <targetDirectory>

above command moves filename1 (either a file or a directory) to the targetDirectory.

class commands.ping.PingCommand(context, log, get_input)

commands.pwd module¶

class commands.pwd.PWDcommand(context, log, get_input)

Returns the location of the present working directory, user currently is residing in the file management system.

Usage: pwd

commands.rm module¶

class commands.rm.RmCommand(context, log, get_input)

Used to delete files and directories. Simple ‘rm’ with a filename will remove that file. Simple ‘rm’ with a directory name will raise an error.

If ‘-rf’ option specified, it will delete the desired directory along with all its constituent directories.

Usage: rm <file_name> rm -rf <directory_name>

recursive_remove(directory)

Recursively removes directories :param directory: Directory model object to search

commands.touch module¶

class commands.touch.TouchCommand(context, log, get_input)

Creates a new file, specified as an argument with the command. Checks for the validity of the file name as well, avoiding duplication in file/directory names.

Usage: touch <file_name>
